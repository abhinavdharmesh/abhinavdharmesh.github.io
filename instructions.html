Ubuntu Performance Optimization

1️⃣ Install Preload (Speeds Up App Launches)

Preload helps apps start faster by predicting what you'll open.

sudo apt install preload -y

2️⃣ Reduce Boot Time (Disable Network Delay Service)

Ubuntu has a network delay service that slows booting. Disabling it helps.

sudo systemctl disable NetworkManager-wait-online.service

3️⃣ Enable ZRAM (Improves RAM Performance)

Since RAM is limited, enabling ZRAM helps reduce swap usage.

sudo apt install zram-tools -y

Do all 3 things then restart:

reboot
