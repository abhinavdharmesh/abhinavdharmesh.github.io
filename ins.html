Ubuntu Performance Optimization

📊 1️⃣ Check and Note Boot Time

Before making changes, check your current boot time:

systemd-analyze

For a detailed breakdown, use:

systemd-analyze blame

Note these times to compare after optimization.

🚀 2️⃣ Install Preload (Speeds Up App Launches)

Preload predicts frequently used applications and loads them into memory.

sudo apt install preload -y

🌐 3️⃣ Disable Network Delay Service (Faster Boot)

Disabling this service can reduce boot delay if your internet takes time to connect.

sudo systemctl disable NetworkManager-wait-online.service

🧠 4️⃣ Enable ZRAM (Improves RAM Performance)

ZRAM creates compressed swap in RAM, reducing pressure on your disk.

sudo apt install zram-tools -y

💻 5️⃣ Optional: Remove Snap & Reinstall Firefox (Non-Snap)

Snap packages are slower to load. Removing Snap can help performance.

sudo systemctl disable snapd.service snapd.socket snapd.seeded.service
sudo apt purge snapd -y

Then install Firefox via Mozilla's APT PPA:

sudo add-apt-repository ppa:mozillateam/ppa
sudo apt update
sudo apt install firefox -y

Prevent Ubuntu from reverting to Snap:

echo '
Package: *
Pin: release o=LP-PPA-mozillateam
Pin-Priority: 1001
' | sudo tee /etc/apt/preferences.d/mozillateam

🎨 6️⃣ Try KDE Plasma (Alternative Desktop Environment)

sudo apt install kde-standard -y

At the login screen, click the gear icon ⚙️ to switch to KDE Plasma session.

If it lags or feels heavy, you can switch back to GNOME or try XFCE later.

🔁 7️⃣ Reboot and Check Boot Time Again

After performing the above steps:

reboot

Then recheck your boot time:

systemd-analyze

Compare with your earlier notes to see the improvement.
