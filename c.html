C Programming Reference Guide

1. The `#include` Directive

In C programming, the `#include` directive is a preprocessor command that tells the compiler to include the contents of a file in the program before the actual compilation process begins. This is typically used to include header files.

#include

Example:

#include <stdio.h>

2. Header Files

Header files in C have a `.h` extension and contain declarations of functions and macros. When you include a header file, you gain access to these declarations, allowing you to use the associated functions and macros in your program.

Common Header Files:

  • stdio.h: Standard Input Output header file.
  • stdlib.h: Standard Library header file.
  • string.h: String handling functions.
  • math.h: Mathematical functions.

3. `#include <stdio.h>`

<stdio.h> is a standard library header file that includes definitions for standard input and output functions like `printf` and `scanf`.

Common Functions in <stdio.h>:

  • printf(): Prints formatted output to the screen.
  • scanf(): Reads formatted input from the keyboard.
  • fopen(), fclose(), fread(), fwrite(): File operations.
  • getchar(), putchar(): Character input and output.

Example Usage:

#include <stdio.h>

int main() {
    printf("Hello, World!\\n");
    return 0;
}

4. Angle Brackets < > vs. Double Quotes " "

Angle Brackets < >:

  • Used for including standard library header files.
  • Tells the preprocessor to look for the header file in the standard system directories.

Double Quotes " ":

  • Used for including user-defined or local header files.
  • Tells the preprocessor to look for the header file in the same directory as the source file first, and then in the standard system directories if not found.

Example of Standard Library Header File:

#include <stdio.h> // Standard library header file

Example of User-Defined Header File:

#include "my_header.h" // User-defined header file

5. Example Program Explained

Here is a simple C program with explanations for each part:

#include <stdio.h> // Includes the standard input-output library

int main() { // The main function is the entry point of a C program
    printf("Hello, World!\\n"); // Calls the printf function from stdio.h to print to the console
    return 0; // Returns 0 to indicate successful execution
}
  • #include <stdio.h>: This line tells the preprocessor to include the standard input-output header file (stdio.h).
  • int main() { ... }: This is the main function where the program starts execution.
  • printf("Hello, World!\\n");: This line calls the printf function (defined in stdio.h) to print "Hello, World!" to the screen.
  • return 0;: This line indicates that the program has finished successfully.

Summary

1. `#include` Directive: Used to include the contents of a header file.

2. Header Files (`.h` files): Contain declarations of functions and macros.

3. `<stdio.h>`: A standard header file for input-output functions.

4. Angle Brackets `< >`: Used for standard library headers.

5. Double Quotes `" "`: Used for user-defined headers.
